# Elite20 Starter Project

这是 Elite20 挑战体系的入门项目模板。

## 项目结构

```
elite20-starter/
├── README.md           # 项目说明
├── REUSE.md            # 拿来主义说明
├── src/                # 源代码目录
│   └── main.py         # 主程序
├── ai_logs/            # AI 对话日志
│   └── chat_log.md     # 聊天记录
├── experiment_logs/    # 实验日志
│   └── exp_01.md       # 实验记录
├── output/             # 产出物
└── portfolio.md        # Portfolio 条目
```

## 快速开始

1. Fork 此仓库
2. 创建你的挑战分支
3. 按照 Elite20 规范提交你的成果

## Elite20 规范要点

- 一句话规则：可复现的产出 + 可追溯的 AI 日志 + 可验证的"拿来"说明
- 审核流程：G1 → G2 → G3
- 三条红线：AI 日志缺失/伪造、伪拿来主义、不可复现
