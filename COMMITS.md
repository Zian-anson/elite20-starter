## GitHub 操作步骤

1. **Fork 仓库**
   ```bash
   # 在 GitHub 上点击 Fork 按钮
   # 或使用命令行
   git clone https://github.com/your-username/elite20-starter.git
   cd elite20-starter
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/my-first-challenge
   ```

3. **提交更改**
   ```bash
   git add .
   git commit -m "feat: 初始化项目结构"
   ```

4. **推送到远程**
   ```bash
   git push origin feature/my-first-challenge
   ```

5. **创建 Pull Request**
   - 在 GitHub 上创建 PR
   - 等待审核
